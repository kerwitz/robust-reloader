How to use

  1. Hit CTRL+L or CMD+L to select the omnibar
  2. Type "r" and press ↹ to activate the extension
  3. Enter one or more intervals and press ↵, for example:
10Reload every ten seconds
10,20,30First reload after ten seconds, 20 seconds, 30 seconds and repeat
10:20Reload every ten minutes and 20 seconds
1:00:00Reload every hour

Supported commands

10,20,20:30One ore more reload intervals
pausePause the reloading
startRestart the reloading
clearClear all set intervals and deactivate the extension
configOpen the options of the extension
aboutOpen this website

Note that the commands can be translated with the extensions _locales. To see if and how they have been translated to your language open up the options panel of the extension.

Hint: although designed for it you are not forced to use the above commands exclusively to control the extension. There is a UI hidden behind the omnibar icon that appears once you have activated the extension with an initial command. Simply click the icon to open the interface.
